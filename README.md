# oneday-onehadis-bot
Bot in Telegram. He sends to all users 1 hadis every day.
Написанный на Python, отправляет каждому своему пользователю по одному рандомному исламскому хадису в день (в 9:00 по мск).
В Базе Данных (SqLite3) взагрузил более 2000+ хадисов, использую достоверный источник HadeethEnc.
